## 三、核心方法论:给庄家行为"定位阶段"

他每篇帖子本质都在回答:**这个庄现在处于哪个阶段?**

```
吸筹 → 拉升 → 加速(逼空) → 派发(出货)
```

| 阶段 | 数据特征 | 他的原话举例 |
|---|---|---|
| **吸筹** | 底部箱体来回,现货悄悄流入,持仓缓慢增 | "箱体构建底部、试盘拉升后回踩" |
| **拉升** | 突破箱体站稳,持仓稳定增,多空比开始走妖 | "突破站稳压力位,聪明钱多头才刚出现雏形" |
| **加速/逼空** | 资费走高、空单爆仓放量、巨鲸空头深套不平 | "加速行情或才刚刚开始,空军非常危险" |
| **派发** | 现货流出、持仓见顶不增价还涨(背离)、资费极端、利空突现 | "现货慢慢出货了""黑客只是出货的理由" |

> **判断对阶段,方向自然就有;判断错阶段,指标看得再准也没用。**

## 四、独立看盘"五问心法"(主菜)

看任何一个币,脑子里走这条流程:

### 第一问:庄在不在?(决定看不看)
看**持仓量**。不增 → 扭头就走,这个币不值得花一秒。
持仓量是"庄家在场证明"。这一问能过滤掉 90% 的噪音。

### 第二问:这是什么性质的局?(决定敢不敢重仓、拿多久)
看**现货 vs 合约**。
- 现货死水 + 合约暴涨 = 纯杠杆逼空局 → **快、猛、塌得也快。轻仓、快进快出。**
- 现货持续真实流入 = 有买盘托底 → **慢、稳、可拿久一点。**

→ 决定你的**仓位和持有周期**。他玩的BEAT/ALLO多是前者,所以他从不"长期持有",全是博弈。

### 第三问:对手盘是谁?(决定有没有肉吃)
看**多空比的结构,不是数字**。问自己:现在亏钱/将被收割的是哪群人?
- 散户人数大量做空 + 大户拿多头 = 逼空金字塔,有肉。
- 散户都在做多 = 小心你自己就是对手盘。

→ 核心:**永远站在要收割别人的那边,别做被收割的那群。**

### 第四问:现在第几阶段?(决定方向和动作)
综合前三问 + 看图,定位在 吸筹/拉升/加速/派发 哪一段。
- 吸筹拉升段 → 找做多机会
- 派发段 / 利空砸盘 → 找做空机会(ZEC、H 就是这个)

### 第五问:什么时候跑?(进场前就要想好)
出场扳机**进场前就定死**,不能边走边想。
- 他的扳机:"资费由正转负 / 极端"
- 你的可以是:资费翻转 / 持仓量见顶回落 / 现货大额流出 / 跌破关键位

> **没想好怎么跑,就不要进。**

## 五、他的两个反直觉"绝活"(最值得内化)

### 绝活一:利空消息一律先问"谁受益"
ZEC增发漏洞、H被黑客盗 —— 他的反应不是恐慌,是"这只是配合出货的理由罢了"。
**心态:任何消息出来,先别信叙事,先看盘面动作(持仓、现货流向)在配合什么。消息是给散户看的台阶。**

### 绝活二:多空比是反指,但要看"谁在哪边"
他不简单看多空比数字,而是拆成**散户人数比 vs 大户持仓比**。
散户拼命做空 + 大户拿多头 = 可收割的金字塔。
普通人看到"70%在做空"会跟着空,他看到的是"70%的人要被收割"。
**永远问一句:亏钱的是谁、对手盘是谁。**

## 七、人机分工:系统做"海选",人做"精判"

```
全市场 400+ 个 U本位合约
        │
   【系统粗筛】← 机器擅长:不知疲倦地扫、算、排序
        │  把 400 个砍到 5-15 个"值得人看一眼"的
        ▼
   【人工精判】← 人擅长:看图形、读庄家意图、定阶段
        │  从中选出 1-3 个真正能讲清逻辑的
        ▼
      下单 + 出场扳机
```

### 机器能做的(→ 交给系统,客观读数)
- 持仓量增减、增速
- 现货流入/流出
- 多空比(散户人数比 vs 大户持仓比)
- 爆仓量(多/空分开)
- 资费数值、符号是否翻转、是否极端
- 市值/持仓比

### 机器做不了的(→ 只能靠人)
- "突破箱体回踩没跌破"等**图形判断**
- "快速下杀没杀到底"等**庄家意图推测**
- "利空只是出货的理由"等**叙事解读**
- "走得很难看,都是针"等**盘面质感**
- 阶段定位的综合直觉

> 系统的定位不是"替你做决定",是"替你把不值得看的票筛掉,让精力只花在该看的票上"。

### 系统筛选逻辑(逻辑层,待定阈值)
```
每隔 N 分钟扫全市场,标记同时满足以下条件的币:

【庄在场】持仓量近 X 日持续增长 且 增速 > 阈值
   AND
【有性质】现货净流向 与 合约成交额 的背离度(纯杠杆局 / 有现货托底,打标签)
   AND
【有对手盘】散户人数多空比 与 大户持仓多空比 出现"金字塔"背离
   AND
【燃料/扳机】资费状态(正常/偏高/极端/刚翻转) + 爆仓量多空占比

→ 输出"庄家行为面板",每个币标注:
   持仓变化 | 现货流向 | 多空比结构 | 资费状态 | 爆仓占比 | 系统猜测的阶段
```
注意:最后一列"阶段"系统只能给规则粗猜,**最终阶段判断由人来定**。

## 八、训练路径(不花钱、不冒险)(2026-06-12 重写)

② 性质:     现货主动买卖__  → 纯杠杆局/有托底
③ 对手盘:   大户多空比__  散户人数比__  爆仓量__
④ 位置:     距30日低点+__%  距高点-__%  近7日+__%
⑤ 我判阶段:  吸筹/拉升/加速/派发,理由一句话
⑥ 假想方向+扳机: ________(讲不清就写"不做")

铁律:**判断卡必须在看他帖子之前、行情走出来之前填**——防事后归因,

### 3. 模拟盘(量化门槛)

"讲不清阶段/对手盘/扳机就不做"保留,另加两条硬规则:

把 120 帖的 (标的, 发帖时间) 对照币安日K回测。跟单模拟:发帖次日(UTC)开盘进场做多,
看 +7 日表现。脚本 `research/sanhe_post_timing.py`,明细 `Temp\sanhe\post_timing.json`。

| 维度 | 全部帖子 | 首次提及 | 后续追踪帖 |
|---|---|---|---|
| **发帖前7日已涨** | 均值 **+84%** / 中位 +51% | +56% / +42% | **+95%** / +54% |
| 跟单+7d 胜率 | **39%** | 45% | 36% |
| 跟单+7d 中位收益 | **-4.5%** | -1.9% | -4.9% |
| 7日内回撤≥20%(5x爆仓) | **55%** | 40% | 61% |
| 7日内回撤≥10%(10x爆仓) | 72% | 60% | 77% |

三个结论:

1. **接棒实锤**——发帖时标的7日内平均已涨84%。"五月就埋伏了"不是虚言,每一帖都发在他的获利区。
2. **平均收益是幸存者假象**——首提+7d均值+6.3%,去掉 SKYAI(+163%)/LAB(+134%)/FIDA(+61%) 三个大赢家后是 **-3.4%**。跟单是买彩票:18%概率吃+30%大肉,中位数亏钱。

> 口径:全部按做多计算(他少数帖做空,如 ZEC/H 派发段,占比小不影响结论)。
> 发帖密度旁证:37天120帖,LAB 一币28帖、BSB 13帖——上车后持续喊单的引流模式。

对系统的反向启示:大赢家在他**首次提及**时就出现(首提胜率45% > 追踪帖36%),
说明"庄家入场"识别确实领先。系统目标不是跟他,是**比他更早**到达同样的候选池(见第七节)。

## 十一、决策漏斗与验证路线(2026-06-12 总结)

### 世界观(一切的前提)

小市值山寨永续的行情**不是价值发现,是庄家的资金操作周期**。价格是结果,资金结构是原因:

> **资金结构 → 庄家意图 → 阶段 → 方向 → 时机**,K线放最后看。
> 散户的顺序恰好相反:先看K线猜涨跌,再找理由。

### 七步决策漏斗(每步一个问题)

```
全市场 ~500 个U本位永续
 ① 庄在不在?   OI日增≥30% 或 成交额≥5倍前日 → 无点火不看,过滤90%
 ② 什么性质?   现货流向 vs 合约爆发 → 纯杠杆局(快进快出)/有托底(可拿久)
                ↑ 决定的不是方向,是仓位和持有周期
 ③ 对手盘是谁?  大户持仓比 vs 散户人数比拆开看;爆仓量小=燃料未耗尽
 ④ 还早吗?     近7日涨幅过滤 + 底部企稳/突破回踩 → 信号来晚了=没有信号
 ⑤ 第几阶段?   吸筹/拉升/加速/派发 → 只做吸筹末~拉升;派发段只离场不做空
 ⑥ 何时跑?     进场前定死扳机(资费翻转/OI见顶/现货流出/破位),讲不清不进
 ⑦ 多大仓?     由②性质定;55%妖币7日内回撤≥20% → 杠杆≤3x
```

三条心法:**看盘先看钱再看价;行情的燃料是对手盘的爆仓单;证据分读数和故事,读数齐全才是交易**。

实证过的四种死法:摸顶做空(他4月底连续爆亏)、信故事开仓(0胜13负)、
追高接棒(发帖时均已涨84%)、杠杆超波动(5x过半爆仓)。

## 十二、因子语义:提示与预警(2026-06-12 讨论稿)

> 核心一句话:**提示看组合,预警看背离**。单因子几乎都是噪音,信号全在组合里。

### 单因子的语义

**1. 持仓量 OI——"庄在不在、筹码在进还是在出"**(注意:OI是多空双向合计,只说明赌桌筹码,不指明方向)
- 提示:OI持续温和增长+价格横盘 = 吸筹指纹;OI单日暴增≥30% = 点火,有局
- 预警:**OI见顶回落+价格还在涨** = 派发背离(聪明钱离场,价格靠惯性)
- 预警:OI暴降+价格瀑布 = 多头连环强平,等OI企稳再看;高位OI滞涨 = 多空僵局,变盘前夜

**2. 价格——"在哪、第几段"**(信息量最低,它是结果;价值在位置和背离)
- 提示:距30日低点涨幅=行程;距前高距离=套牢盘/真空区
- 预警:**价格独自创新高**——OI/量/现货任一不跟,每多一个不跟,派发概率加一档

**3. 资金费率——"杠杆情绪温度+谁在给谁付钱"**(最被低估,因为它直接是钱)
- 提示:深负费率+价格不跌 = 空军重仓被套硬扛(LAB -1%/8h),逼空燃料,做多还收租
- 提示:费率中性+OI涨 = 现货驱动,杠杆未过热,行情更健康
- 预警:极端正费率 = 散户杠杆多头拥挤,拉升末端指纹;**费率符号翻转** = 主导换边(他的出场扳机,值得继承)

**4. 合约成交量——"换手热度、局的真假"**
- 提示:成交额倍数爆发(5x~70x) = 点火确认;拉升后缩量回调 = 健康洗盘
- 预警:**天量滞涨** = 对倒出货;**无量拉升** = 控盘虚拉,塌时也无量(他空ZEREBRO的理由)

**5. 现货——"局的性质:真金白银还是纯杠杆"**(性质判定器,决定仓位和周期,不决定方向)
- 提示:现货持续净流入+合约涨 = 有托底,可拿久、回调敢补
- 提示:现货死水+合约暴增 = **纯杠杆逼空局**,快猛塌得快,轻仓快进快出
- 预警:**现货大额流出+价格还在涨** = 派发实锤中的实锤(现货出了就是真走了,比OI背离更硬)

### 组合矩阵(OI×价格主轴,费率/现货/量做修正)

| OI | 价格 | 修正项 | 解读 | 动作 |
|---|---|---|---|---|
| ↑ | 横盘 | 现货流入、费率中性 | **吸筹** | 进观察池,等点火 |
| ↑↑ | ↑ | 量倍增、费率温和负 | **拉升启动**(最佳进场窗) | 核对位置后进场 |
| ↑ | ↑↑ | 费率转极端正 | **加速/逼空尾段** | 只持不加,扳机上膛 |
| **↓** | ↑ | 现货流出 | **派发** | 离场,不做空 |
| ↓↓ | ↓↓ | 费率剧烈翻负 | 多头强平瀑布 | 空仓,等OI企稳 |
| 平 | ↓缓 | 现货悄悄流入 | 洗盘后二次吸筹? | 回观察池 |

| 因子结论 | 等级 | 说明 |
|---|---|---|
